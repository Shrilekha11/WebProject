Algorithms in Steganography using Weighted & Matching Techniques.
Platform : MATLAB R2010b

Steganography is the art and science of hiding communication. The focus was mainly on Image Steganography, where cover media is an image and secret message is in the form of text. The popular method for imasge steganography is LSB insertion mechanism, where secret message bits are directly inserted into the LSB of the image pixel byte. Here two new algorithms were developed: Weighted Technique and Matching Technique. Weighted technique is based on LSB but here we are taking weighted average of bits to compare it with secret message bits and then storing in the LSB of image pixel byte. In the second algorithm i.e. Matching Technique, secret message bits and image pixel bytes are divided into pairs of groups of three consecutive bits. Then matching is done between pair of secret message bits against image pixel byte’s pair. If it matches then the respective position is stored in LSB bits i.e.01, 10 and 11. If none of the pairs matches with the secret message bits, then 00 is inserted to the LSB bits of image byte. These algorithms were finally compared and analyzed for the results by plotting graph of PSNR values versus message length (KB). We found that new algorithms are efficient when compared to LSB technique.
